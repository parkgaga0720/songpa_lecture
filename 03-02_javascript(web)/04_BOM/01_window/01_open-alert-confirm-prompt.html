open alert confirm prompt 메소드

브라우저 환경

        자바스크립트가 돌아가는 플랫폼은 호스트라고 불린다.
        호스트 환경이 웹 브라우저일때 사용할 수 있는 기능은 아래와 같다.
        window
           ㄴ DOM (document...)
           ㄴ BOM (location, navigator ...)
           ㄴ JavaScript (object, array, function ...)

open

        window.open(url, name, params) 메소드로 새창을 열 수 있다.
        url : 새창에 로드할 url이다.
        name : 새창의 이름으로 해당 이름을 가진 창이 있으면 그 안에서 열리고 그렇지 않으면 새창이 열린다
        params : 새창의 설정을 숨표로 구분하여 문자열로 전달한다.

alert

        window.alert(message) 메소드는 확인 버튼을 가지며 메세지를 지정할 수 있는 경고 대화 상자를 띄운다.

confirm

        window.confirm(message) 메소드는 확인과 취소 두 버튼을 가지며 메세지를 지정할 수 있는 모달 대화 상자를 띄운다.
        반환갑은 확인을 누를시 true, 취소를 누르거나 esc키를 누르면 false이다.

prompt

        window.prompt(message,default)메소드는 사용자가 텍스트를 입력할 수 있도록 안내하는
        선택적 메세지를 갖고 있는 대화상자를 띄운다.
        default:텍스트 입력 필드에 기본으로 채워 넣을 문자열이다
        반환 값이 확인을 누를 시 사용자가 입력한 문자열이며 취소를 누르거나 esc키를 누르면 null이다.
