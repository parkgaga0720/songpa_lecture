font-size 속성 : 글씨의 크기를 변경할 때 사용한다.

  • 고정크기 px 테스트
  • 가변크기 em 테스트
  • 가변크기 % 테스트
  • 가변크기 pt 테스트

font-weight 속성 : 글씨를 두께를 표현할 때 사용한다.

  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 가늘게
  • 원래 굵기보다 더 굵게

font-variant 속성 : 작은 대문자로 변경할 때 사용한다.

Hello World

Hello World

font-style 속성 : 글씨의 기울임을 지정할 때 사용한다.

  • italic 기울임 글꼴
  • oblique 기울임 글꼴

font 속성 : 글꼴 관련 스타일을 한번에 지정할 때 사용한다.

  • 글꼴 관련 스타일 테스트1
  • 글꼴 관련 스타일 테스트2
