텍스트 스타일

color - 텍스트의 색상을 지정할 수 있다

색상값 참조 사이트

  • 색상명으로 지정
  • 16진수 값으로 지정
  • rgb로 지정
  • rgba로 지정
  • hsl로 지정
  • hsla로 지정

text-decoration 속성 텍스트에 줄을 긋거나 줄을 없앨때 사용하는 속성

  • 하이퍼링크 밑줄 없애기
  • 텍스트 영역 아래 줄 긋기
  • 텍스트 영역 중간 줄 긋기
  • 텍스트 영역 위에 줄 긋기

text-transform 속성 영문 텍스트의 대소문자 변환시 사용한다

  • none
  • capitalize : 첫글자만 대문자
  • uppercase : 대문자
  • lowercase : 소문자
