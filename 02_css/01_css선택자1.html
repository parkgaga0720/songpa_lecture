CSS 기본 선택자

선택자(selector)란?
특적한 태그를 선택할 때 사용하는 기능으로, 태그를 선택하여 원하는 스타일과 기능을 적용할수 있다.

모든 선택자

문서 안의 모든 태그를 선택할 대 전체 선택자를 이용할 수 있다.

  • test 1
  • test 2
  • test 3

태그 선택자

문서 내에 같은 태그를 모두 선택할때 사용한다.

아이디 선택자

문서 내에 해당하는 아이디 속성의 값을 가진 태그만 선택할 때 사용한다.

  1. test 1
  2. test 2
  3. test 3
  4. test 4
  5. test 5

클래스 선택자

문서 내에서 여러개의 태그를 동일한 클래스명으로 지정하여 선택할 때 사용한다

  • 클래스 선택자 테스트1
  • 클래스 선택자 테스트2
  • 클래스 선택자 테스트3
  • 클래스 선택자 테스트4
  • 클래스 선택자 테스트5
