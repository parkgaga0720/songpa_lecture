폼관련태그

form태그는 html사용자가 입력할 수 있는 양식을 제공ㅇ하는 태그이다
form태그 내의 input태그를 통해 사용자가 입력한 정보를 서버로 넘기는 역할을 한다
action 속성 : 폼의 입력된 값들을 전송받을 서버의 클래스 명을 입력한다. method 속성 : get/post 방식으로 전송방식을 지정한다. (get 방식은 url에 넘어가는 데이터가 보임) (post 방식은 url에 넘어가는 데이터가 보이지 않음)

submit 버튼 눌렀을 때 action에 지정한 경로로 메소드에 지어한 방식에 따라 input 태그에 입력한 값을 전달한다

필드셋의 제목을 작성하는 부분이다


필드셋으로 묶은 영역별로 구분된다.

type = "password" 비밀번호를 입력할 수 있는 텍스트 상자이다

type = "search", "url", "email, "tel" 겉모습은 텍스트와 비슷하지만 각각의 정보에 맞게 분화된 기능을 제공하는 텍스트 상자이다
